# 3043. Find the Length of the Longest Common Prefix

## Problem Statement

You are given two arrays:

```cpp
arr1
arr2
```

A **prefix** of a number is formed by starting from the leftmost digit.

Examples:

```cpp
123 -> prefixes are:
1
12
123
```

We need to find:

```cpp
The maximum length of a common prefix
between any number in arr1 and any number in arr2.
```

---

# Example

## Input

```cpp
arr1 = [1, 10, 100]
arr2 = [1000]
```

---

## Explanation

Common prefixes:

```cpp
1     -> length 1
10    -> length 2
100   -> length 3
```

So the answer is:

```cpp
3
```

---

# Approach 1: Hash Table

# Intuition

The key observation:

A prefix can be generated by repeatedly removing the last digit.

Example:

```cpp
12345
```

Prefixes:

```cpp
12345
1234
123
12
1
```

We can:

1. Store all prefixes from `arr1`
2. Check prefixes of numbers in `arr2`
3. Find the longest match

---

# Key Idea

For every number in `arr1`:

```cpp
while (num > 0) {
    store(num);
    num /= 10;
}
```

This generates all prefixes.

---

# Example

## arr1

```cpp
[100]
```

Generated prefixes:

```cpp
100
10
1
```

Stored in hash set.

---

## arr2

```cpp
1000
```

Check:

```cpp
1000 -> not found
100  -> found
```

Prefix length:

```cpp
3
```

---

# Algorithm

## Step 1: Build Prefix Set

For every number in `arr1`:

* Store current number
* Remove last digit
* Continue until number becomes 0

---

## Step 2: Match Prefixes

For every number in `arr2`:

* Keep removing last digit
* Stop when prefix exists in set
* Compute prefix length

---

# Hash Table Solution (C++)

```cpp
class Solution {
public:
    int longestCommonPrefix(vector<int>& arr1, vector<int>& arr2) {

        unordered_set<int> arr1Prefixes;

        // Store all prefixes from arr1
        for (int val : arr1) {

            while (!arr1Prefixes.count(val) && val > 0) {

                arr1Prefixes.insert(val);

                // Remove last digit
                val /= 10;
            }
        }

        int longestPrefix = 0;

        // Check prefixes for arr2
        for (int val : arr2) {

            while (!arr1Prefixes.count(val) && val > 0) {

                val /= 10;
            }

            if (val > 0) {

                int digits = log10(val) + 1;

                longestPrefix = max(longestPrefix, digits);
            }
        }

        return longestPrefix;
    }
};
```


```cpp
O(m * log10(M) + n * log10(N))
```

Why?

Each division by:

```cpp

```

removes one digit.

A number with:

```cpp
d digits
```

takes:

```cpp
O(d)
```

operations.

---

## Space Complexity

```cpp
O(m * log10(M))
```

Because all prefixes are stored in hash set.

# Trie Algorithm

## Insert

For every digit:

* move/create node
* continue traversal

---

## Search

For every digit:

* if child exists:

  * increment length
  * continue
* otherwise:

  * stop

---

# Trie Solution (C++)

```cpp
class TrieNode {
public:

    TrieNode* children[10];

    TrieNode() {

        for (int i = 0; i < 10; i++) {
            children[i] = nullptr;
        }
    }
};

class Trie {
public:

    TrieNode* root;

    Trie() {
        root = new TrieNode();
    }

    // Insert number into Trie
    void insert(int num) {

        TrieNode* node = root;

        string numStr = to_string(num);

        for (char digit : numStr) {

            int idx = digit - '0';

            if (!node->children[idx]) {
                node->children[idx] = new TrieNode();
            }

            node = node->children[idx];
        }
    }

    // Find longest matching prefix
    int findLongestPrefix(int num) {

        TrieNode* node = root;

        string numStr = to_string(num);

        int len = 0;

        for (char digit : numStr) {

            int idx = digit - '0';

            if (node->children[idx]) {

                len++;

                node = node->children[idx];
            }
            else {

                break;
            }
        }

        return len;
    }
};

class Solution {
public:

    int longestCommonPrefix(vector<int>& arr1,
                            vector<int>& arr2) {

        Trie trie;

        // Insert arr1 into Trie
        for (int num : arr1) {
            trie.insert(num);
        }

        int longestPrefix = 0;

        // Search arr2
        for (int num : arr2) {

            int len = trie.findLongestPrefix(num);

            longestPrefix = max(longestPrefix, len);
        }

        return longestPrefix;
    }
};
```
